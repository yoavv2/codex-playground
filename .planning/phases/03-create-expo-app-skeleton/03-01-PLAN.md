## tasks

### Task 1: Lock Phase 03 decisions and create context file

1. Inspect Farfield workspace conventions (`package.json`, scripts, TypeScript/lint patterns).
2. Decide and document:
   - mobile package location and name (`farfield/apps/mobile`, `@farfield/mobile`)
   - app routing approach (Expo Router)
   - persisted settings approach (`expo-secure-store` for token + durable storage for non-secret profile values)
3. Resolve the deferred SSE package decision for Phase 03 and document rationale in a new context file.
4. Create `.planning/phases/03-create-expo-app-skeleton/03-CONTEXT.md` with:
   - locked decisions
   - constraints inherited from Phase 02
   - plan split boundaries (`03-01`, `03-02`, `03-03`)
   - open watch items for Phase 04

Deliverable:
- `03-CONTEXT.md` exists and can be consumed by `03-02` and `03-03` without reopening decisions.

### Task 2: Scaffold Expo mobile workspace package

1. Create `farfield/apps/mobile` as a workspace package using Expo + TypeScript.
2. Add package scripts for:
   - development (`start`, `ios`, `android`, `web`)
   - static checks (`typecheck`, `lint`)
3. Add required config files:
   - `app.json`
   - `babel.config.js`
   - `tsconfig.json`
   - entrypoint/config required by Expo Router
4. Update repository ignore rules and docs for mobile build/runtime artifacts as needed (for example `.expo`).
5. Ensure root workspace tooling can target the mobile app with `bun run --filter @farfield/mobile ...`.

Deliverable:
- `apps/mobile` is scaffolded, dependency setup is coherent, and workspace scripts resolve.

### Task 3: Validate foundation and record handoff to 03-02

1. Run foundation checks for the new package.
2. Update `.planning/STATE.md` after execution:
   - set next action to execute `03-02-PLAN.md`
   - keep Phase 03 as active phase
3. Capture implementation results in:
   - `.planning/phases/03-create-expo-app-skeleton/03-01-SUMMARY.md`
   - include explicit boundary handoff notes for `03-02`

Deliverable:
- Foundation is complete and next step is unambiguous (`03-02`).
